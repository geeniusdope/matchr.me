# Domain name ideas for the platform

Suggestions for a mutual-interest, low-rejection meeting platform (work email, match only when both add each other). **Availability changes constantly** — check with a registrar (Namecheap, Porkbun, Google Domains, etc.) before deciding.

---

## Mutual / both-ways

| Name                  | Notes                                    |
| --------------------- | ---------------------------------------- |
| mutualmatch.com / .io | Clear and on-concept.                    |
| bothways.io / .app    | Short, “both ways” = mutual.             |
| bothadded.com         | Literal: both added each other.          |
| mutualadd.com / .io   | Add each other → mutual.                 |
| twowaymatch.com / .io | Two-way match.                           |
| doubleopt.in          | Double opt-in; .in can read as “opt in.” |
| weboth.io             | “We both” — we both added.               |

## “I like you, you like me” wordplay

Names that play on the phrase “I like you, you like me” (mutual interest).

### Direct phrase (or close)

| Name                   | Notes                                                                              |
| ---------------------- | ---------------------------------------------------------------------------------- |
| ilikeulikeme.com / .me | “I like u like me” — compact. **ilikeulikeme.me** doubles the “me” (phrase + .me). |
| ulikemeilikeu.com      | “U like me, I like u” — same idea, different order.                                |
| welikeeachother.com    | “We like each other” — clear, a bit long.                                          |
| webothlike.com         | “We both like [each other]” — short.                                               |
| bothlike.com / .io     | Even shorter; reads as “both like.”                                                |
| youlikemeilikeyou.com  | Full phrase; long for a domain.                                                    |

### “Like me back” / reciprocal

| Name                 | Notes                                |
| -------------------- | ------------------------------------ |
| likemeback.com / .me | “Like me back” = they like you back. |
| likemeback.io        | Same; .me fits “me” in the phrase.   |
| likeuback.com        | “Like u back.”                       |

### Wordplay / brandable

| Name              | Notes                                                 |
| ----------------- | ----------------------------------------------------- |
| likelikeme.io     | “Like, like me” (like = crush, like = similar to me). |
| ulikeme.com / .me | “You like me” — one half of the pair.                 |
| ilikeu.com        | “I like u” — other half; often taken.                 |
| mutualike.com     | Mutual + like.                                        |

**Strong fits:** **ilikeulikeme.me**, **likemeback.com** or **.me**, **webothlike.com**, **bothlike.com** or **.io**.

## Tips

- Prefer **.com** for broad recognition; **.io** or **.app** are fine for a product brand.
- Avoid names that imply “we’ll tell you who liked you” — they conflict with the no-rejection principle.
- Short, easy to spell and say; avoid sounding overly “dating” if you want workplace/professional use.
